## 🔎 이진수 ❓

### 📌 비트(bit) : 컴퓨터에서 자료를 표현하는 최소 단위 0 또는 1

### 📌 바이트(byte) : 8 bit

### 📌 이진수 : 컴퓨터에서 표현되는 자료의 형태<br>                0000 1010 (이진수) => 10 (십진수)

### 📌 8 진수 : 이진수를 3비트씩 모아서 표현

### 📌 16 진수 : 이진수를 4비트씩 모아서 표현
